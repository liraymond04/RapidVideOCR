    简介 link

    • 视频硬字幕提取，自动生成对应srt | txt文件。
    • 支持字幕语言：中文 | 英文 （其他可以支持的语言参见：支持语种列表)
    • 优势如下：
      • 提取更快：与VideoSubFinder软件结合使用，提取关键字幕帧更快。
      • 识别更准：采用RapidOCR作为识别库。
      • 使用更方便：pip直接安装即可使用。
    • 如果有帮助到您的话，请给个小星星⭐。
